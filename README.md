## envirment
*   python = 3.6
*   numpy = 1.14.3
*   pandas = 0.23.0
*   tensorflow-gpu= 1.1.15
*   keras = 2.2.4

## install
Install anaconda and add anaconda into envirment path.
```
conda create --name myenv python=3.6
```

clone this github and activate VM
```
activate myenv
```

and run this 

``` 
pip install -r requirements.txt
```
